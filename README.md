# rollup-single-import-cost

This packages provides an utility which bundles a dummy project with a single import to estimate the size of a given import.
Calculates size of tree-shaken, minified code.

Uses:
- Rollup with:
- commonjs plugin
- node-resolve plugin
- terser?

Usage:
```js

```
